# Eksperimen_SML_GilangPutraFirmansyah
AutoFlow ML
## 1. Perkenalan Dataset

### Sumber Dataset
Dataset yang digunakan adalah **Credit Card Default Dataset** yang diperoleh dari:

UCI Machine Learning Repository  
https://archive.ics.uci.edu/ml/datasets/default+of+credit+card+clients

### Deskripsi Dataset
Dataset ini berisi data klien kartu kredit di Taiwan pada tahun 2005.  
Dataset digunakan untuk memprediksi apakah seorang nasabah akan mengalami gagal bayar pada bulan berikutnya.

### Jumlah Data
- Total sampel: 30.000 baris
- Total fitur: 23 fitur numerik
- Target: `default.payment.next.month`

### Jenis Masalah
Binary Classification:
- 0 → Tidak gagal bayar
- 1 → Gagal bayar

### Tujuan Eksperimen
Melakukan eksperimen preprocessing data sebagai tahap awal sebelum:
- Training model
- AutoML
- Monitoring & deployment
